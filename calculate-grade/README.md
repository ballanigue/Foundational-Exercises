# 📝Grade Calculator
Focuses on boolean and conditional functions. A working code that determines the letter grade.

## How to Use?
1. Download the file here: [CalculateGrade](https://github.com/ballanigue/Foundational-Exercises/blob/main/calculate-grade/CalculateGrade.py)
2. Run the code in whichever apps you use (Windows Powershell, VS Code, etc...)
3. Upon running the code, a menu will appear and it will ask the user what they want to do

## Contents of Grade Calculator
- Determines the letter grade based on the scores given by user
- Range of A+ to F-
- Consist of performance task, test, seatwork, and recitation
