# Foundational-Exercises 🏛️
This repository includes all of the collections of small projects and exercises that are required during my SHS Programming Elective. It covers basic functions, loops, and conditional logic.
## 📝 What it includes... 
### [Lottery Simulator](https://github.com/ballanigue/Foundational-Exercises/tree/main/lottery-simulator)
This covers basic and random functions, loops, logic, and many more. A simulation application that handles a random sequence of numbers.
#### _IMPORTANT_!
Install colorama in your Termal by pasting the code there (pip install colorama). Otherwise, the code will not run as it lacks the necessary files.
#### Contents of the Lottery Simulator
- Generating Tickets, the user is up to how many times they wish to generate.
- Draw a Winning Number
- Display the Number of Winners in the Set of Tickets Generated.

### [Calculate Grade](https://github.com/ballanigue/Foundational-Exercises/tree/main/calculate-grade)
Mainly focuses on boolean and condition functions. A code that asks the user for score and calculates its letter grade.
#### Contents of the Calculate Grade
- Determines the letter grade based on the scores given by user
- Range of A+ to F-
- Consist of performance task, test, seatwork, and recitation

### [Core Exercises](https://github.com/ballanigue/Foundational-Exercises/tree/main/core-exercise)
Programming workout and exercises to learn how the code works. Full list of small codes/programs:
- Factorial
- Loops
- Plotting
- Debugging
